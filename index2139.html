Features

Lightweight

CustomRP barely uses system resources: under 15MB of disk space and RAM.

Tray Minimization

The app will work quietly in the background and will show up when you want to.

Run on Startup

The app can automatically start with Windows, so your status will show right away.

Multilingual

The app supports multiple languages. If yours is missing, you can help translate the app.
